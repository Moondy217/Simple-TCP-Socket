💡 **소켓 통신 구현**

1. 클라이언트와 서버가 통신해서 받을 파일은 temp2 폴더에 저장될 예정 파일은 이미 temp1 폴더에 저장되어 있음
2. temp2 폴더에서 cmd 실행
3. 클라이언트 측에서 서버에 연결 요청. 서버는 연결 수락 후 temp1에 미리 저장해 둔 파일명 전송
4. 클라이언트는 전송받을 파일 이름을 서버에게 전송, 서버는 해당 파일을 클라이언트에게 전송
